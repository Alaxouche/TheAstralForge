# Using The List

## Starting The List

Open `ModOrganizer.exe` from your Extrasolar Containment Protocol installation folder and launch the game through the executables dropdown — select the **SFSE** launcher. As the modlist can be quite large, depending on your PC it may take some time to load the game. **THIS IS NORMAL, DON'T PANIC!**

{: .note }
You can create a desktop shortcut from the shortcuts dropdown in Mod Organizer 2.

{: .warning }
> Do not launch the game through Steam. This will launch vanilla Starfield without the modlist.

{: .warning }
> Add the list to your antivirus exceptions list, as it may interfere with certain files.

## Do Not Click Unlock!

Mod Organizer 2 locks your mods, load order, and other executables while running an executable. This is to prevent you from making changes while the game is running. Clicking unlock may corrupt your game experience.

## Configuring MCM / Settings

Where applicable, MCMs and in-game settings have been pre-configured. However, you can adjust them to your personal preferences. Be careful when modifying pre-set configurations — note down any default values before changing them in case you need to revert.

{: .warning }
> Some settings are loaded automatically on new game start. Make sure to wait for all background processes to finish before making changes.

## New Game vs. Existing Save

Extrasolar Containment Protocol is designed to be started on a **new game**. Loading an existing vanilla save is not supported and may cause unexpected issues or crashes.
