# Pre-Installation

## Necessary Tweaks

Before installing the modlist, here are some things you must do:  
There is no workaround for a pirated copy of Starfield. If you ask for support with anything pirated, you will be banned.

{: .warning}
> **WARNING**
>
> If your Steam library is in Program Files, refer to [this](https://github.com/LostDragonist/steam-library-setup-tool/wiki/Usage-Guide) guide to move it elsewhere. Do not install to default Windows folders (Desktop, Documents, Downloads, etc.) as this will cause issues.

In the Windows API (with some exceptions), the maximum length for a file path is 260 characters. Keep your installation paths short to avoid issues.

{: .note }
The list cannot be installed in the same folder that Wabbajack is in. They must be completely separated.

Here is an example of a well-configured installation folder:
- `C:\Games\ExtrasolarContainmentProtocol\`
- `C:\Games\ExtrasolarContainmentProtocol\Downloads\`

## Accounts and Websites

So the Wabbajack installation happens without problems, you must connect or create accounts for the following websites:
- [Nexus Mods](https://www.nexusmods.com/) (Premium Recommended for faster installation)

## Preparing Starfield

1. Open Steam and ensure Starfield is installed and up to date.
2. Launch the game at least once through Steam to generate the initial configuration files.
3. Close the game completely before proceeding.

{: .warning }
> Do not have any mods installed in Starfield's data folder before running Wabbajack. This can cause conflicts and installation failures.
