# Pre-Installation

## Necessary Tweaks

Before installing the modlist, here are some things you must do:
There is no workaround for a pirated copy of Starfield. If you ask for support with anything pirated, you will be banned.

{: .warning}
> **WARNING**
>
> If your Steam library is in Program Files, refer to [this](https://github.com/LostDragonist/steam-library-setup-tool/wiki/Usage-Guide) guide to move it elsewhere. Do not install to default Windows folders (Desktop, Documents, Downloads, etc.) as this will cause issues.

In the Windows API (with some exceptions), the maximum length for a file path is 260 characters. Keep your installation paths short to avoid issues.

{: .note }
The list cannot be installed in the same folder that Wabbajack is in. They must be completely separated.

Here is an example of a well-configured installation folder:
- `C:\Games\ExtrasolarContainmentProtocol\`
- `C:\Games\ExtrasolarContainmentProtocol\Downloads\`

## Accounts and Websites

For the Wabbajack installation to proceed without problems, you must connect or create accounts for the following websites:
- [Nexus Mods](https://www.nexusmods.com/) (Premium Recommended for faster installation)

## Preparing Starfield

1. Open Steam and ensure Starfield is installed and up to date.
2. Launch the game at least once through Steam to generate the initial configuration files.
3. Close the game completely before proceeding.

{: .warning }
> Do not have any mods installed in Starfield's data folder before running Wabbajack. This can cause conflicts and installation failures.

# Pré-Installation

## Modifications Nécessaires

Avant d'installer la modlist, voici quelques choses que vous devez faire :
Il n'y a pas de solution de contournement pour une copie piratée de Starfield. Si vous demandez de l'aide pour quoi que ce soit de piraté, vous serez banni.

N'installez pas dans les dossiers Windows par défaut (Bureau, Documents, Téléchargements, etc.) car cela causera des problèmes.

Dans l'API Windows (avec quelques exceptions), la longueur maximale d'un chemin de fichier est de 260 caractères. Gardez vos chemins d'installation courts pour éviter les problèmes.

{: .note }
La liste ne peut pas être installée dans le même dossier que Wabbajack. Ils doivent être complètement séparés.

Voici un exemple de dossier d'installation bien configuré :
- `C:\Games\ExtrasolarContainmentProtocol\`
- `C:\Games\ExtrasolarContainmentProtocol\Downloads\`

## Comptes et Sites Web

Pour que l'installation Wabbajack se déroule sans problème, vous devez vous connecter à ou créer des comptes sur :
- [Nexus Mods](https://www.nexusmods.com/) (Premium recommandé pour une installation plus rapide)

## Préparer Starfield

1. Ouvrez Steam et assurez-vous que Starfield est installé et à jour.
2. Lancez le jeu au moins une fois via Steam pour générer les fichiers de configuration initiaux.
3. Fermez complètement le jeu avant de continuer.

{: .warning }
> N'ayez aucun mod installé dans le dossier data de Starfield avant de lancer Wabbajack. Cela peut causer des conflits et des échecs d'installation.
