# Using The List

## Starting The List

Open `Mod Organizer.exe` and launch **Ghost of the Grid Launcher** through the executables. This is a custom executable of cyberpunk.exe with custom arguments to skip intro and deploy RedMod. Loading may take some time. **THIS IS NORMAL — DON'T PANIC!**

{: .note }
You can create a desktop shortcut in the shortcuts dropdown.

{: .warning }
> Do not launch the game through Steam or GOG. This will launch vanilla Cyberpunk 2077.

{: .warning }
> Add the list to your antivirus exceptions list.

## Do Not Click Unlock!

Mod Organizer locks your mods and load order while running an executable. Clicking unlock may corrupt your game experience.

## Switching Profiles

This modlist comes with three experiences:
- **Ghost of the Grid - Performance**: No custom traffic vehicles, no high-resolution faces/NPCs, no improved LODs, lighter settings.
- **Ghost of the Grid - Standard**: Base experience with high quality visuals and medium settings for balanced performance.
- **Ghost of the Grid - Ultra**: High quality visuals with ultra settings for the best visual experience.

Switch between them using the profile dropdown in the upper left of MO2.

## Updating

- Always check the Changelog before updating
- **Backup your saves** or start fresh after updates
- Wabbajack will delete all files not part of the modlist when updating

# Utiliser la Liste

## Démarrer la Liste

Ouvrez `Mod Organizer.exe` et lancez **Ghost of the Grid Launcher** via les exécutables. C'est un exécutable personnalisé de cyberpunk.exe avec des arguments pour passer l'intro et déployer RedMod. Le chargement peut prendre du temps. **C'EST NORMAL — NE PANIQUEZ PAS !**

{: .note }
Vous pouvez créer un raccourci bureau dans le menu déroulant des raccourcis.

{: .warning }
> Ne lancez pas le jeu via Steam ou GOG. Cela lancera Cyberpunk 2077 vanilla.

{: .warning }
> Ajoutez la liste aux exceptions de votre antivirus.

## Ne Cliquez Pas sur Déverrouiller !

Mod Organizer verrouille vos mods et l'ordre de chargement pendant l'exécution. Cliquer sur déverrouiller peut corrompre votre expérience de jeu.

## Changer de Profil

Cette modlist propose trois expériences :
- **Ghost of the Grid - Performance** : Pas de véhicules personnalisés, pas de visages/PNJ haute résolution, pas de LODs améliorés, paramètres allégés.
- **Ghost of the Grid - Standard** : Expérience de base avec visuels haute qualité et paramètres moyens pour un équilibre performance/visuels.
- **Ghost of the Grid - Ultra** : Visuels haute qualité avec paramètres ultra pour la meilleure expérience visuelle.

Changez de profil via le menu déroulant en haut à gauche de MO2.

## Mise à Jour

- Consultez toujours le journal des modifications avant de mettre à jour
- **Sauvegardez vos sauvegardes** ou commencez une nouvelle partie après les mises à jour
- Wabbajack supprimera tous les fichiers ne faisant pas partie de la modlist lors des mises à jour
