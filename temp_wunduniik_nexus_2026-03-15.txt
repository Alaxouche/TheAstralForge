Wunduniik is a flagship Skyrim Special Edition collection focused on high-end visuals, immersive gameplay, and broad content depth.
Goal: transform Skyrim into a modern, large-scale experience while keeping a coherent world and progression loop.

---

## Main Features

- Large visual overhaul: upgraded textures, lighting, weather, and scene fidelity.
- Gameplay overhauls: combat, progression, and systems tuned for long campaigns.
- Expanded content: new encounters, equipment, and world activity density.
- Quality of life layer: better interface flow, fixes, and smoother day-to-day play.
- Curated mega-list design: extensive but maintained as a cohesive package.

---

## Before You Start

- Skyrim Special Edition 1.6.1170 required.
- Anniversary Edition required.
- Legit Steam copy required.
- SSD installation strongly recommended.
- Nexus Mods account required (Premium recommended for speed).
- Add game and modlist folders to antivirus exclusions.

---

## Quick Install

1. Install required runtimes (.NET Runtime and Visual C++ x64).
2. Install Wabbajack in a dedicated folder.
3. Search for Wunduniik in Wabbajack.
4. Select clean install/download paths outside protected Windows folders.
5. Run full installation and wait for completion.
6. Launch only through Mod Organizer profile/launcher.
7. Follow the complete guide before first gameplay session.

## Quick Troubleshooting

- Download failures: rerun Wabbajack and retry.
- CTD after major update: verify save compatibility and update notes.
- Heavy-script setup issues: allow first load/setup to fully finish before testing changes.
- Performance mismatch: use the recommended profile where applicable.
