# Using The List

## Starting The List

Open `Mod Organizer.exe` and launch **No Man's Sky - Explorer Launcher** through the executables. This is a custom executable of NMS.exe. There will be a warning message when launching regarding mods — simply click to skip to the main menu. **THIS IS NORMAL — DON'T PANIC!**

{: .note }
You can create a desktop shortcut in the shortcuts dropdown.

{: .warning }
> Do not launch the game through Steam or GOG. This will launch vanilla No Man's Sky.

{: .warning }
> Add the list to your antivirus exceptions list.

## Do Not Click Unlock!

Mod Organizer locks your mods and load order while running an executable. Clicking unlock may corrupt your game experience.

## Updating

- Always check the Changelog before updating
- **Backup your saves** or start fresh after updates
- Wabbajack will delete all files not part of the modlist when updating

# Utiliser la Liste

## Démarrer la Liste

Ouvrez `Mod Organizer.exe` et lancez **No Man's Sky - Explorer Launcher** via les exécutables. C'est un exécutable personnalisé de NMS.exe. Un message d'avertissement apparaîtra au lancement concernant les mods — cliquez simplement pour passer au menu principal. **C'EST NORMAL — NE PANIQUEZ PAS !**

{: .note }
Vous pouvez créer un raccourci bureau dans le menu déroulant des raccourcis.

{: .warning }
> Ne lancez pas le jeu via Steam ou GOG. Cela lancera No Man's Sky vanilla.

{: .warning }
> Ajoutez la liste aux exceptions de votre antivirus.

## Ne Cliquez Pas sur Déverrouiller !

Mod Organizer verrouille vos mods et l'ordre de chargement pendant l'exécution. Cliquer sur déverrouiller peut corrompre votre expérience de jeu.

- Consultez toujours le journal des modifications avant de mettre à jour
- **Sauvegardez vos sauvegardes** ou commencez une nouvelle partie après les mises à jour
- Wabbajack supprimera tous les fichiers ne faisant pas partie de la modlist lors des mises à jour
