## Reporting an Issue

If during your playthrough you encounter any bug or incompatibility that disturbs your gameplay or immersion, please head over to my Discord server and share the bug in the dedicated channel so I can try to fix it in a future update. We work together to make this modlist always better!

### Share your crash log

If you crash using the latest version of Wunduniik and want to report it, here is where you can find the crash log:

**Crash Log Location:**
`Documents/My Games/Skyrim Special Edition/SKSE/crash-202x-xx-xx-xx-xx-xx.log`

## INI Configuration

Some INI settings may be unoptimised. Open BethINI and make sure the settings are like this:
- Shadow Resolution: 2048
- Ambient Occlusion: Use either this or the ENB version. Do not have both turned on.
- Remove Shadows: Not recommended, but available if needed.

## Mods Configuration

The modlist is exclusively in 2K. You can lower clutter retextures to 1K if needed. You can also change some Fomod settings.

Mod tags used in the list:
- **Basic Fomod**: Regular fomod with only add-ons and other options.
- **Patches Fomod**: May contain patches and add-ons.
- **Unpublished**: Self-explanatory.
- **Output**: Files generated for the list. Regenerate if you change something that affects them.

### Saving Your Mod Changes

Before updating, rename any mods you added by adding the prefix `[NoDelete]`. These are found on the left side of MO2 under the Mod Name column.

You can copy the `loadorder.txt` from your profiles folder within the Wunduniik installation. Save the copy outside of the Wunduniik folder so it is not deleted during an update.

# Questions & Réponses

## Signaler un Problème

Si au cours de votre partie vous rencontrez un bug ou une incompatibilité qui perturbe votre gameplay ou votre immersion, rendez-vous sur mon serveur Discord et partagez le bug dans le canal dédié afin que je puisse essayer de le corriger dans une future mise à jour. Nous travaillons ensemble pour améliorer continuellement cette modlist !

### Partager votre journal de crash

Si vous avez un crash avec la dernière version de Wunduniik et souhaitez le signaler, voici où trouver le journal de crash :

**Emplacement du journal de crash :**
`Documents/My Games/Skyrim Special Edition/SKSE/crash-202x-xx-xx-xx-xx-xx.log`

## Configuration des INIs

Certains paramètres INI peuvent être non optimisés. Ouvrez BethINI et vérifiez ces réglages :
- Résolution des ombres : 2048
- Occlusion Ambiante : Utilisez soit celle-ci, soit la version ENB. N'activez pas les deux.
- Supprimer les ombres : Non recommandé, mais disponible si nécessaire.

## Configuration des Mods

La modlist est exclusivement en 2K. Vous pouvez abaisser les retextures d'objets à 1K si nécessaire. Vous pouvez également modifier certains paramètres Fomod.

Balises de mods utilisées dans la liste :
- **Basic Fomod** : Fomod classique avec uniquement des extensions et autres options.
- **Patches Fomod** : Peut contenir des correctifs et des extensions.
- **Unpublished** : Non publié.
- **Output** : Fichiers générés pour la liste. Régénérez-les si vous modifiez quelque chose qui les affecte.

### Sauvegarder Vos Modifications de Mods

Avant de mettre à jour, renommez les mods que vous avez ajoutés en ajoutant le préfixe `[NoDelete]`. Ils se trouvent sur le côté gauche de MO2 dans la colonne Nom du Mod.

Vous pouvez copier le fichier `loadorder.txt` depuis votre dossier de profils dans l'installation de Wunduniik. Sauvegardez la copie en dehors du dossier Wunduniik pour qu'elle ne soit pas supprimée lors d'une mise à jour.
