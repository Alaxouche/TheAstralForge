# Wabbajack Installation

## Installation

Once you have completed pre-installation, download the latest version of Wabbajack and place it in a folder such as `C:\Games\Wabbajack`. Do not place it in program files, on your desktop or in your downloads folder. I recommend placing it on an SSD as it will work quicker on there.

Downloading and installing Wunduniik can take a while depending on your internet connection and computer. Follow the next steps to download and install Wunduniik properly:

1. Open Wabbajack and click on **Browse Modlists**.
2. Go to the Search field and type **Wunduniik**.
3. Make sure **Show NSFW** is not checked.
4. Press the download button on Wunduniik and wait for it to download.
5. Set the installation folder to somewhere like `C:\Games\Wunduniik`. Do not install it to your desktop or downloads folder.
6. The download location does not need to be on an SSD but it makes installing a bit faster.
7. Press the play button to begin.
8. Go and pet your nearest fluffy animal whilst Wabbajack does its thing. Alternatively read through this readme again.
9. The installation is now complete. You can move on to the next section.

# Installation Wabbajack

## Installation

Une fois la pré-installation terminée, téléchargez la dernière version de Wabbajack et placez-le dans un dossier comme `C:\Games\Wabbajack`. Ne le placez pas dans les fichiers programmes, sur votre bureau ou dans votre dossier de téléchargements. Je recommande de le placer sur un SSD car il fonctionnera plus rapidement.

Le téléchargement et l'installation de Wunduniik peuvent prendre un certain temps selon votre connexion internet et votre ordinateur. Suivez les étapes suivantes pour télécharger et installer Wunduniik correctement :

6. L'emplacement de téléchargement n'a pas besoin d'être sur un SSD, mais cela accélère un peu l'installation.
7. Appuyez sur le bouton lecture pour commencer.
8. Allez caresser l'animal le plus proche pendant que Wabbajack fait son travail. Vous pouvez aussi relire ce guide.

## Problems With Installation

It is possible that you may encounter an error with Wabbajack when installing. Some common issues are listed below.

**Could not download x:**
Big files can fail to download due to connection issues. You can either run Wabbajack again or download the file manually. If you decide to manually download it, make sure to place it in the same place as the other downloads.

**A French translation mod is failing to download:**
Some files use special characters such as "ç" and "ä" that can fail to download if your PC can't handle those. To fix this:
- Set your PC to English (United States)
- Disable "Use Unicode UTF-8"
- Restart the download. Once done you can revert those settings.

## Problèmes d'Installation

Il est possible que vous rencontriez une erreur avec Wabbajack lors de l'installation. Voici quelques problèmes courants.

**Impossible de télécharger x :**
Les gros fichiers peuvent échouer à télécharger en raison de problèmes de connexion. Vous pouvez relancer Wabbajack ou télécharger le fichier manuellement. Si vous téléchargez manuellement, placez le fichier au même endroit que les autres téléchargements.

**Un mod de traduction française échoue au téléchargement :**
Certains fichiers utilisent des caractères spéciaux comme « ç » et « ä » qui peuvent échouer à télécharger si votre PC ne les prend pas en charge. Pour corriger cela :
- Définissez votre PC sur Anglais (États-Unis)
- Désactivez « Utiliser Unicode UTF-8 »
- Relancez le téléchargement. Une fois terminé, vous pouvez rétablir ces paramètres.
