# Using The List

## Starting The List

Open `Mod Organizer.exe` and launch **Wunduniik Launcher** through the executables. It should be already automatically selected. This is a custom executable of SKSE. As the modlist is quite big, it may take some time to load the game depending on your PC. **THIS IS NORMAL — DON'T PANIC!**

{: .note }
You can create a desktop shortcut in the shortcuts dropdown.

{: .warning }
> Do not launch the game through Steam. This will launch vanilla Skyrim.

{: .warning }
> Add the list to your antivirus exceptions list as it may otherwise stop it from working.

## Do Not Click Unlock!

Mod Organizer locks your mods, load order, and other executables while running an executable. This is to prevent the user from making changes while the game is running. Clicking unlock may corrupt your game experience.

## Configuring MCM

All MCMs have been pre-configured via Settings Loader mods, however you can change settings if you don't like them. Settings Loader automatically loads the presets at game launch. **BE CAREFUL!** If you change MCM settings, there is no way to recover the originals. Make sure to note what was set by default before making changes.

{: .warning }
> Failure to wait for MCMs to finish loading can cause issues later on and will require you to start a new save.

# Utiliser la Liste

## Démarrer la Liste

Ouvrez `Mod Organizer.exe` et lancez **Wunduniik Launcher** via les exécutables. Il devrait être déjà sélectionné automatiquement. C'est un exécutable personnalisé de SKSE. Comme la modlist est assez grande, le chargement du jeu peut prendre un certain temps selon votre PC. **C'EST NORMAL — NE PANIQUEZ PAS !**

{: .note }
Vous pouvez créer un raccourci bureau dans le menu déroulant des raccourcis.

{: .warning }
> Ne lancez pas le jeu via Steam. Cela lancera Skyrim vanilla.

{: .warning }
> Ajoutez la liste aux exceptions de votre antivirus car il pourrait l'empêcher de fonctionner.

## Ne Cliquez Pas sur Déverrouiller !

Mod Organizer verrouille vos mods, l'ordre de chargement et les autres exécutables pendant qu'un exécutable est en cours d'exécution. Cela empêche l'utilisateur de faire des modifications pendant que le jeu tourne. Cliquer sur déverrouiller peut corrompre votre expérience de jeu.

## Configuration des MCM

Tous les MCM ont été préconfigurés via des mods Settings Loader, mais vous pouvez modifier les paramètres si vous ne les aimez pas. Settings Loader charge automatiquement les préréglages au lancement du jeu. **ATTENTION !** Si vous modifiez les paramètres MCM, il est impossible de récupérer les originaux. Notez ce qui était défini par défaut avant de faire des changements.

{: .warning }
> Ne pas attendre que les MCM finissent de se charger peut causer des problèmes et vous obligera à recommencer une nouvelle sauvegarde.
